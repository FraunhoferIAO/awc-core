Adaptive Web Components Examples

Component Implementation

  • Variants using Template Elements
    demonstrates the implementation of an AdaptiveComponent with multiple AdaptiveVariants each using a specific <template> element.
  • Variants that create Document Fragments
    demonstrates the implementation of AdaptiveVariants that create DocumentFragments on-the-fly in their template getters.

Component Usage

  • Context Scoping with Profile Stores
    demonstrates how multiple ProfileStores can be used in one document by attaching them to different parts of the DOM.
  • CSS Custom Properties in Variants
    demonstrates how to use CSS custom properties (a.k.a. CSS variables) in components.

Context Style

  • Document-wide Context Styles
    demonstrates how the context-style element can be used to define document-wide context-dependent style rules.
  • Context Styles in Component Templates
    demonstrates how the context-style element can be used for context-dependent styling in AdaptiveVariant templates.
